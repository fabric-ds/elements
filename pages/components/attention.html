Accessibility

For assistive technologies to read contents of the displayed Attention element, set `role="tooltip"` on the relevant text element nested in `f-attention`. Elements with the tooltip role should be referenced through the use of `aria-describedby` before or when the tooltip is displayed. The `aria-describedby` attribute is on the target element, not on `f-attention`.
