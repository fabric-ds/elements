Toast

The toast is intended to be used programmatically. JavaScript APIs are provided to create, update and remove toasts from a page while managing things like placement on the page for you.

Toast is a bit different from other packages in Fabric Elements. You need to import functions from the package and call them as needed.

Import functions for working with toasts:

import { toast, removeToast, updateToast } from '@fabric-ds/elements/toast'; Create a new toast giving it a message: toast('This is a toast');
Create a new toast giving it a message and some options and then get back an id that can be used later to remove or edit the toast: const id = toast('This is a toast', { type: 'warning' }); Update an existing toast by id: updateToast(id, { text: 'This is a toast' });

Options

Type of toast: type — default success (success, warning, error, info)

toast('message goes here', { type: 'warning' })

Text content: text — default undefined

const id = toast('message goes here'); updateToast({ id, text: 'change the message' });

Auto removal: duration — default 5000ms

toast('message goes here', { duration: 2500 })

Closable: canclose — default true

toast('message goes here', { canclose: false, duration: 2500 })
